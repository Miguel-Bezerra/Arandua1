Esse é um site para compartilhamento de histórias e contos folclóricos entre os usuário, ainda em fase de testes, existem alguns erros.

Para entrar no site, baixe as pastas do repositório e as abra no VS Code. Após isso, clique com o botão direito em "Backend" (na pasta), e vá em "Abrir terminal integrado". Após abrir o terminal, digite "npm start", isso habilitará o backend. Quando isso for concluído, se você tiver a extensão "Live Server", você deve baixá-la. Para isso, abra o menu de extensões do Visual e procure por "Live Server". Após a instalação da extensão, clique com o botão direito em "login.html", na pasta Tela_login, por sua vez na pasta Frontend, e selecione a opção "Abrir com Live Server", o que deve te redirecionar para o site.

Ou, abra o link a seguir no navegador: aranduaa.netlify.app
